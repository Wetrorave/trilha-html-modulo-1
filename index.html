O que são Tags HTML?

Início

As tags são os principais elementos utilizados para estruturar e organizar o conteúdo de uma página da web. São parte integral do HTML, a linguagem de marcação usada para criar sites e páginas da internet. As tags são como blocos de construção, que informam ao navegador como exibir cada parte do conteúdo.

Como funcionam?

Início

Cada tag HTML é composta por um nome entre colchetes angulares. Normalmente, ela possui uma abertura e um fechamento. A tag de fechamento é semelhante à de abertura, mas com uma barra (/) antes do nome. Veja um exemplo:

<p>Este é um parágrafo.</p>

Nesse caso:

  • <p> é a tag de abertura;
  • </p> é a tag de fechamento;
  • O texto entre as duas tags é o conteúdo do parágrafo.

Tipos de Tags Comuns

Início
  1. Estruturais:
    • <html>: Envolve todo o código da página.
    • <head>: Contém informações sobre a página (título, estilos, etc.).
    • <body>: Contém o conteúdio visível da página.
  2. Textuais:
    • <strong>: Texto em negrito (com significado de importância).
    • <mark>: Texto marcado como por um marca-texto.
    • <font>: Texto estilizado (com o atributo style).
    • <del>: Texto traçado.
  3. Listas:
    • <ul>: Lista não ordenada.
    • <ol>: Lista ordenada.
    • <li>: Item de lista.
  4. Imagens e links:
    • <img>: Exibe uma imagem (não precisa de fechamento).
    • <a>: Cria um link (com o atributo href).
  5. Tabelas e formulários: Usam diversas tags específicas, como <table>, <tr>, <td>, <form>, <input>, entre outras.

Tags vazias

Início

Algumas tags não têm conteúdo interno e, portanto, não precisam de tag de fechamento. Elas são chamadas de tags vazias ou auto-fechadas. Gosto de indicá-las usando uma barra depois do nome, como:

<img src="imagem.jpg" alt="Descrição da imagem"/>
<br/> <!-- Quebra de linha -->
<hr/> <!-- Linha horizontal -->

Conclusão

Início

As Tags HTML são essenciais para definir o conteúdo e a estrutura de uma página na web. Ao entender como usá-las corretamente, é possível criar páginas bem organizadas, acessíveis e funcionais. Elas formam a base da web e são o primeiro passo para quem deseja aprender desenvolvimento front-end.
